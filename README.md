# fricon

Data collection automation framework.

## Documentation

[Github Pages](https://kahojyun.github.io/fricon/)

## Development

### Requirements

- Stable Rust toolchain
- Node.js 22+
- Protobuf compiler
- Sqlite3
- diesel_cli
- `uv` and `pnpm`

### Local setup

```console
python3 scripts/setup-dev.py
```
